### ✅ 4. Добавлен выбор действия при клике на рыбу

**Новая функциональность:**

При клике на рыбу в режиме просмотра открывается модальное окно с выбором:

1. **🏆 Сделать трофеем**
   - Показываются все рыбы из группы
   - Можно выбрать конкретные экземпляры
   - Отображается вес и длина каждой рыбы
   - Подтверждение создания трофеев

2. **💰 Выбрать для продажи**
   - Показываются все рыбы из группы
   - Можно выбрать какие именно продать
   - Визуальная индикация выбранных рыб
   - Подтверждение продажи

### ✅ 5. Детальный просмотр и выбор рыб

**Если у вас 60 карасей:**
- Открывается сетка со всеми 60 карасями
- Каждая карточка показывает:
  - Изображение рыбы
  - Вес (⚖️ X.XX кг)
  - Длину (📏 XX.X см)
- Можно выбрать любое количество
- Выбранные рыбы подсвечиваются зеленым

### API endpoints

Используются:
- `GET /api/inventory/grouped` - сгруппированный инвентарь
- `GET /api/inventory` - детальный список всех рыб
- `POST /api/make-trophy` - создание трофея
- `POST /api/sell-bulk` - продажа рыбы

## Как использовать

### Для пользователя:

1. **Открыть лавку**
2. **Кликнуть на любую рыбу**
3. **Выбрать действие:**
   - Сделать трофеем
   - Выбрать для продажи
4. **Выбрать конкретные рыбы** из списка
5. **Подтвердить** действие
