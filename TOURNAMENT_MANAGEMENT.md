# Управление турнирами артелей

## Что изменено:

### 1. Увеличен лимит отображаемых артелей
- **Было:** 20 артелей
- **Стало:** 100 артелей (максимум 200)
- Теперь в списках "АРТЕЛИ" и "РЕЙТИНГ" отображаются все созданные артели

### 2. Добавлено время в создание турниров
- **Было:** Только дата (начало и конец)
- **Стало:** Дата + время (datetime-local)
- Теперь можно создавать турниры на точное время, например, с 14:30 до 14:35

### 3. Турнир виден сразу после создания
- **Было:** Турнир появлялся только после начала
- **Стало:** Турнир виден с момента создания (фаза "upcoming")
- Три фазы турнира:
  - **upcoming** - Турнир ещё не начался
  - **active** - Турнир идёт
  - **grace** - Турнир завершён (итоги видны ещё сутки)

### 4. Добавлена функция удаления турниров
- Новая кнопка "🗑️ УДАЛИТЬ ТУРНИР" на вкладке "ТУРНИР" (только для админов)
- Турнир удаляется полностью из базы данных
- Не нужно ждать сутки после окончания турнира

## Как создать тестовый турнир на пару минут:

1. Откройте webapp, перейдите на вкладку "АРТЕЛИ"
2. Нажмите кнопку "🏆 СОЗДАТЬ ТУРНИР АРТЕЛЕЙ"
3. Заполните форму:
   - **Название:** Тестовый турнир
   - **Начало (дата и время):** Выберите текущую дату и время (например, 2026-06-07T14:30)
   - **Конец (дата и время):** Выберите время через 2-3 минуты (например, 2026-06-07T14:33)
4. Нажмите "СОЗДАТЬ"
5. Появится подтверждение, и автоматически появится вкладка "ТУРНИР"

## Как удалить турнир:

1. Перейдите на вкладку "ТУРНИР"
2. Прокрутите список до конца
3. Нажмите кнопку "🗑️ УДАЛИТЬ ТУРНИР"
4. Подтвердите удаление
5. Турнир будет полностью удалён из базы данных

## Проблемы и решения:

### Вкладка "ТУРНИР" не появляется после создания
**Причины:**
1. Время на сервере отличается от вашего локального (сервер использует UTC)
2. Турнир создан с неправильным временем

**Решение:**
- Проверьте консоль браузера (F12) на наличие логов "Tournaments refreshed"
- Убедитесь, что время турнира указано корректно
- Учитывайте, что сервер работает в UTC (возможно, нужно скорректировать время)

### "Турнир недоступен" на вкладке ТУРНИР
**Причины:**
1. Турнир закончился более суток назад
2. Турнир не загрузился

**Решение:**
- Обновите страницу (перейдите на другую вкладку и вернитесь)
- Проверьте консоль на ошибки
- Удалите старый турнир и создайте новый

## Где хранятся турниры:

Турниры хранятся в таблице `clan_tournaments` в базе данных SQLite/PostgreSQL:

```sql
-- Структура таблицы
CREATE TABLE clan_tournaments (
    id SERIAL PRIMARY KEY,
    title TEXT NOT NULL,
    starts_at TIMESTAMP NOT NULL,
    ends_at TIMESTAMP NOT NULL,
    created_by INTEGER NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## API endpoints:

- **GET** `/api/guilds` - Получить список артелей (лимит по умолчанию 100)
- **GET** `/api/guilds/tournaments` - Получить список турниров
- **POST** `/api/guilds/tournaments/create` - Создать турнир
- **POST** `/api/guilds/tournaments/delete` - Удалить турнир (только админ)

## Ручное удаление турнира через SQL (если нужно):

Если нужно удалить турнир напрямую из базы данных:

```sql
-- Посмотреть все турниры
SELECT id, title, starts_at, ends_at FROM clan_tournaments ORDER BY id DESC;

-- Удалить турнир по ID
DELETE FROM clan_tournaments WHERE id = 1;

-- Удалить все турниры (осторожно!)
DELETE FROM clan_tournaments;
```

## Примечания:

- Только пользователь с ID `793216884` может создавать и удалять турниры
- При удалении турнира все связанные данные (результаты, статистика) остаются в базе
- Турниры видны с момента создания до конца + 1 сутки
- Сервер использует UTC время, учитывайте это при создании турниров
