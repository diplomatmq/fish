## Как применить исправления

### Вариант 1: Автоматически (рекомендуется)

Просто перезапустите бота - все исправления применятся автоматически:

```bash

# Затем запустите снова
python bot.py
```

При запуске автоматически:
- Создастся таблица `user_fish_stats`
- Добавятся все необходимые индексы
- Выполнятся миграции BIGINT

### Вариант 2: Вручную через скрипт

```bash
# 1. Проверьте текущее состояние
python scripts/check_fixes.py

# 2. Примените оптимизации
```
