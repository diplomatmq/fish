# Шпаргалка по событиям на локациях

## 🎯 Главное

### Механика
- ✅ События **глобальные** - для всей локации
- ✅ **Все игроки** на локации получают эффект
- ✅ Проверка при **каждой рыбалке** (любого игрока)
- ✅ События **мигрируют** между локациями

### Ограничения
- ⚠️ На локации - **только 1 событие**
- ⚠️ Тип события - **только на 1 локации**

## 📊 События

### 🐟 Нерест
```
Эффект:  +20% шанс рыбы (не мусор)
Шанс:    0.1% при рыбалке
Время:   45-90 минут
CD:      6 часов
Частота: ~1-2 раза/час (активная локация)
```

### ☠️ Убийство
```
Эффект:  Только 1 вид рыбы
Шанс:    0.03% (очень редко!)
Время:   30-60 минут
CD:      12 часов
Частота: ~1 раз в 3-5 часов (активная локация)
```

### 🐠 Стайный инстинкт
```
Эффект:  +10% вес за рыбу (макс +50%)
Шанс:    0.08%
Время:   60-120 минут
CD:      8 часов
Частота: ~1-3 раза/час (активная локация)

Особенность: цепочки индивидуальны!
- Событие глобальное
- Каждый игрок ведет свою цепочку
- Сброс при ловле другой рыбы
```

## 🔄 Жизненный цикл

```
1. Игрок рыбачит → проверка шанса
2. Событие появилось! ✅
3. ВСЕ игроки на локации видят событие
4. Длительность: 30-120 минут
5. Событие закончилось
6. Cooldown: 6-12 часов
7. После CD событие может появиться на ДРУГОЙ локации
```

## 💡 Примеры

### Сценарий 1: Популярная локация
```
"Море" (100 рыбалок/час)
→ Нерест появляется каждые 1-2 часа
→ Много игроков ловят с бонусом
→ Соревнование за улов
```

### Сценарий 2: Тихая локация
```
"Глухое озеро" (10 рыбалок/час)
→ События редкие (~раз в 10 часов)
→ Меньше конкуренции
→ Можно "фармить" в одиночку
```

### Сценарий 3: Миграция
```
10:00 "Море"  → Нерест (90 мин)
11:30         → конец → cooldown 6ч
17:30         → CD истек
18:00 "Река"  → Нерест (60 мин)
19:00         → конец → cooldown 6ч
...
```

## 🛠️ Команды

### Для игроков
```
/weather  - показать события на локации
/events   - показать события на локации (альтернатива)
```

### Для админов
```
/events start spawn Море      - запустить Нерест
/events start murder Река     - запустить Убийство
/events start school Пруд     - запустить Стаю
/events stop spawn Море       - остановить
```

## 🎮 Стратегия игроков

### Нерест
- Рыбачить активно
- Использовать кормушки
- Ловить много рыбы

### Убийство
- Идеально для целевой рыбы
- Не важна наживка
- Гарантированный вид

### Стайный инстинкт
- Выбрать правильную наживку
- Ловить только целевой вид
- Пропускать другую рыбу
- Цель: набить цепочку до 5

## 🔒 Что НЕЛЬЗЯ

❌ Два события на одной локации
```
"Море": Нерест ✅
"Море": Убийство ❌ - локация занята
```

❌ Одно событие на двух локациях
```
"Море": Нерест ✅
"Река": Нерест ❌ - событие занято
```

## ✅ Что МОЖНО

✅ Разные события на разных локациях
```
"Море":  Нерест ✅
"Река":  Убийство ✅
"Пруд":  Стая ✅
```

## 📈 Влияние активности

| Рыбалок/час | Частота Нереста | Частота Убийства |
|-------------|-----------------|------------------|
| 100+        | ~1-2 ч          | ~3-5 ч           |
| 20-50       | ~5-10 ч         | ~15-25 ч         |
| <10         | редко           | очень редко      |

## 🎯 Итог

**Для игроков:**
- Следить за `/weather`
- Путешествовать за событиями
- Делиться инфой в чате

**Для сообщества:**
- "Горячие точки" на локациях
- Концентрация игроков
- Социальный аспект

**Баланс:**
- События редкие → ценные
- Мигрируют → разнообразие
- Глобальные → соревнование
