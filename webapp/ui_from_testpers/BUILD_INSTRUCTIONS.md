# Инструкция по сборке UI

## Что было изменено

### 1. Уменьшена кнопка "ПРОДАТЬ"
- Теперь кнопка компактная и не занимает много места
- Размер: `padding: 8px 16px; font-size: 14px;`

### 2. Исправлен цвет текста в панели выбора
- Текст "Выбрано: X" теперь белый с тенью
- Фон панели более контрастный: `rgba(0,0,0,0.95)`

### 3. Панель выбора остается на месте при скролле
- Используется `position: fixed; bottom: 80px;`
- Панель всегда видна над нижней навигацией

### 4. Добавлен выбор действия при клике на рыбу
При клике на рыбу в режиме просмотра открывается модальное окно с выбором:
- 🏆 **Сделать трофеем** - выбрать какие рыбы станут трофеями
- 💰 **Выбрать для продажи** - выбрать какие рыбы продать

### 5. Детальный выбор рыб
- Показываются все рыбы из группы (например, все 60 карасей)
- Можно выбрать конкретные экземпляры
- Отображается вес и длина каждой рыбы
- Визуальная индикация выбранных рыб

## Как собрать

```bash
cd webapp/ui_from_testpers

# Установить зависимости (если еще не установлены)
npm install

# Собрать production версию
npm run build

# Или запустить dev сервер для разработки
npm run dev
```

## Структура изменений

### Файлы:
- `src/ui/shopScreen.ts` - основная логика лавки
- `src/style.css` - стили интерфейса

### Новые функции:
- `showFishDetailModal(item)` - показ деталей группы рыб
- `makeTrophies(ids)` - создание трофеев из выбранных рыб

### Новые стили:
- `.fish-detail-grid` - сетка для отображения всех рыб
- `.fish-detail-card` - карточка отдельной рыбы
- `.fish-action-buttons` - кнопки действий

## API endpoints

Используются следующие endpoints:
- `GET /api/inventory/grouped` - получить сгруппированный инвентарь
- `GET /api/inventory` - получить детальный список всех рыб
- `POST /api/make-trophy` - создать трофей из рыбы
- `POST /api/sell-bulk` - продать рыбу

## Тестирование

1. Откройте лавку в приложении
2. Проверьте что кнопка "ПРОДАТЬ" компактная
3. Нажмите на любую рыбу - должно открыться модальное окно
4. Выберите "Сделать трофеем" или "Выбрать для продажи"
5. Выберите конкретные рыбы из списка
6. Подтвердите действие
7. Проверьте что панель выбора остается на месте при скролле

## Troubleshooting

### Если изменения не применились:
```bash
# Очистить кэш и пересобрать
rm -rf dist
npm run build
```

### Если ошибки TypeScript:
```bash
# Проверить типы
npm run type-check
```

### Если не работает в production:
- Убедитесь что файлы из `dist/` копируются на сервер
- Проверьте что nginx/сервер отдает новые файлы
- Очистите кэш браузера (Ctrl+Shift+R)
