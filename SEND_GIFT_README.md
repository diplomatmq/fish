# Система отправки подарков /send

## Описание
Команда `/send` позволяет игрокам отправлять подарки друг другу через личные сообщения с ботом.

## Стоимость
- **25 звезд** за каждую отправку (независимо от типа и количества подарка)

## Доступные типы подарков

### 1. Монеты 💰
- Отправитель вводит количество монет
- Требование: у отправителя должно быть достаточно монет
- Монеты списываются у отправителя и добавляются получателю

### 2. Удочки 🎣
- Выбор из списка всех удочек в игре
- Требование: 
  - У отправителя должна быть эта удочка
  - У отправителя должно быть достаточно монет для покрытия стоимости удочки
- Получатель получает удочку через систему стакания (если у него уже есть временная удочка, использования суммируются)

### 3. Наживки 🪱
- Выбор локации → выбор наживки → ввод количества
- Требование:
  - У отправителя должно быть достаточно наживки
  - У отправителя должно быть достаточно монет для покрытия стоимости наживки
- Наживка списывается у отправителя и добавляется получателю

### 4. Сети 🕸️
- Отправитель вводит количество сетей
- Требование: у отправителя должно быть достаточно сетей
- Сети списываются у отправителя и добавляются получателю

### 5. Одежда 👕
- Выбор из списка всей одежды в игре
- Требование:
  - У отправителя должна быть эта одежда
  - У отправителя должно быть достаточно бриллиантов для покрытия стоимости одежды
- Одежда добавляется получателю

## Процесс отправки

1. Игрок вызывает `/send` в личных сообщениях с ботом
2. Выбирает тип подарка из меню
3. Выбирает конкретный предмет (для удочек/наживок/одежды) или вводит количество (для монет/сетей)
4. Вводит @username или ID получателя
5. Оплачивает 25 звезд через Telegram Stars
6. Подарок передается получателю
7. Оба игрока получают уведомления

## Ограничения

- Команда работает **только в личных сообщениях** с ботом
- Нельзя отправить подарок самому себе
- Получатель должен иметь профиль в игре
- У отправителя должно быть достаточно ресурсов для отправки

## Отмена

В любой момент до оплаты можно использовать `/cancel` для отмены процесса отправки подарка.

## Технические детали

### Файлы
- `send_gift_system.py` - основная логика системы подарков
- `bot.py` - обработчики команд и callback'ов
- `database.py` - функции работы с базой данных

### Callback patterns
- `send_select_{type}` - выбор типа подарка
- `send_rod_{rod_name}` - выбор удочки
- `send_rods_page_{page}` - пагинация удочек
- `send_clothing_{code}` - выбор одежды
- `send_clothing_page_{page}` - пагинация одежды
- `send_bait_loc_{location}` - выбор локации для наживок
- `send_bait_{bait_name}` - выбор наживки
- `send_baits_page_{location}_{page}` - пагинация наживок
- `send_cancel` - отмена отправки
- `send_back_main` - возврат в главное меню

### Payment payload
- Format: `send_gift_{sender_id}_{recipient_id}`
- Pre-checkout проверка ресурсов
- Successful payment выполняет передачу

### Уведомления
- Отправитель получает подтверждение с деталями передачи
- Получатель получает уведомление о подарке (если возможно отправить DM)

## Примеры использования

### Отправить 1000 монет
1. `/send`
2. Выбрать "💰 Монеты"
3. Ввести `1000`
4. Ввести `@username` или `123456789`
5. Оплатить 25 звезд

### Отправить удочку
1. `/send`
2. Выбрать "🎣 Удочки"
3. Выбрать нужную удочку из списка
4. Ввести `@username` или `123456789`
5. Оплатить 25 звезд

### Отправить наживку
1. `/send`
2. Выбрать "🪱 Наживки"
3. Выбрать локацию
4. Выбрать наживку
5. Ввести количество
6. Ввести `@username` или `123456789`
7. Оплатить 25 звезд
